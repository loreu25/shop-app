# Shop App
Одностраничное приложение с фронтендом на React и бэкендом на ASP.NET MVC.

## Структура
- `frontend/` — React-приложение.
- `backend/` — ASP.NET MVC бэкенд.

## Запуск фронтенда
1. `cd frontend`
2. `npm install`
3. `npm start`

## Запуск бэкенда
1. `cd backend`
2. Открой `BackEnd.sln` в Visual Studio.
3. Запусти проект.
